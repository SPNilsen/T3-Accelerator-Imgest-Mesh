1 Introduction

This report details the comprehensive process undertaken for Project ARMOR at JnJ-MEDTECH, where we develop a robust image classification system using a Baseline Multi‑Input CNN. Leveraging an end-to-end pipeline—from data ingestion and preprocessing (with careful splitting of visual and UV components) to training with k-fold cross-validation—we systematically evaluate model performance and document our findings. This framework not only establishes a solid baseline for image classification but also sets the stage for future comparative analyses with more advanced architectures.
